# Product 001 Build Evidence - PlayPatch Kids

Status: First local prototype

## Built

- Static PWA prototype under `apps/playpatch`.
- Node local server for preview.
- Activity plan builder.
- Activity deck.
- Activity player with supplies, steps, safety note, timer, favorites, and
  completion patches.
- Starter activity pack and two parent-gated premium pack previews.
- Optional parent settings for sound cues and calm endings.
- Parent gate and privacy guardrail panel.

## Commands

```bash
pnpm playpatch:check
pnpm playpatch
```

## Current URL

```text
http://localhost:4273/
```

## Release Notes

This is a web/PWA prototype only. It is not yet ready for App Store submission.
Before mobile release, Factory must add accessibility checks, content review,
privacy labels, App Store screenshots, parent-gate hardening, and a native
wrapper decision.
