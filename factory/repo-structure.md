# Factory Repository Structure

Status: Draft baseline for Factory 1.0

## Local Root

Factory lives under:

```text
~/BuildPod/
```

## Top-Level Folders

```text
factory/   Internal operating system for BuildPodFactory.
products/  Future product workspaces.
modules/   Reusable product modules.
research/  Market, product, technical, and tool research.
evidence/  Cross-product evidence archives.
archive/   Retired drafts, exports, and old baselines.
```

## Factory Folder

```text
factory/
  agents/
  governance/
  operations/
  security/
  templates/
```

## Rule

Downloads and Desktop are temporary only. Controlled Factory files must live under BuildPod.
