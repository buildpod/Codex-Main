# Reusable Module Roadmap

Status: Draft baseline for Factory 1.0

## Purpose

Prioritize reusable modules that future Factory products can share.

## Priority Order

1. Auth and identity.
2. Tenant/workspace.
3. Roles and permissions.
4. Audit/evidence ledger.
5. File upload and storage.
6. Import engine.
7. Reporting/export.
8. Notifications.
9. Billing/subscription.
10. AI gateway.
11. Admin console.

## Module Rule

Reusable modules must have a module contract before being reused across products.

## Reuse Decision

Before reuse, Factory must decide:

- Is the module mature enough?
- Does the product need customization?
- Are security/data assumptions compatible?
- Is version/change history clear?
