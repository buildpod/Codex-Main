# UX Quality Standard

Status: Draft baseline for Factory 1.0

## Purpose

Define the minimum UX bar for products created by Factory.

## UX Rules

- Build the actual product workflow, not a marketing shell.
- Keep enterprise tools clear, calm, and task-focused.
- Use predictable navigation.
- Use responsive layouts for laptop and mobile where relevant.
- Show useful empty, loading, success, and error states.
- Keep text readable and inside containers.
- Do not hide critical actions.
- Use accessible labels and keyboard-friendly controls where practical.
- Avoid decorative complexity that does not help the task.

## Required Review

Before release, core workflows need UX review for:

- First-use clarity.
- Task completion.
- Error recovery.
- Mobile/laptop layout.
- Visual polish.
- Accessibility basics.
