# Decision Rights Matrix

Status: Draft baseline for Factory 1.0

## Purpose

Define who may recommend, accept, or approve Factory decisions.

| Decision | AI May Recommend | Codex/Claude May Implement | Human Owner Approval |
| --- | --- | --- | --- |
| Tool install | Yes | No | Required |
| Factory policy edit | Yes | Yes, after request | Required for baseline |
| Product idea selection | Yes | No | Required |
| Product build start | Yes | No | Required |
| Architecture choice | Yes | Yes after approval | Required for high impact |
| Security policy | Yes | No final approval | Required |
| Secret handling | No secrets in AI | No | Required |
| Release approval | Yes | No | Required |
